# notepad

上の**キーの登録**でのOpen editorは`src/main/res/values/strings.xml` のファイルを開くと出てくる．

つぎの `activity_main.xml` に戻り，「Nameにある...」は `activity_main.xml` の`Design` 右上の `Attributes` を押す．
